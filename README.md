# FAQ Chatbot using NLP

## Description
This project implements a simple FAQ chatbot using Natural Language Processing techniques.

## Features
- Answers frequently asked questions
- Uses TF-IDF vectorization
- Uses cosine similarity for matching
- Terminal based chatbot

## Technologies Used
- Python
- Scikit-learn
- NLTK

## Installation

pip install -r requirements.txt

## Run

python chatbot.py
