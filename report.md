# Project Report

## Project Title
FAQ Chatbot using NLP

## Objective
To develop a chatbot capable of answering frequently asked questions.

## Technologies Used
- Python
- Scikit-learn
- NLTK

## Methodology
1. Store FAQs in JSON format.
2. Convert questions into vectors using TF-IDF.
3. Compare user query with existing questions.
4. Use cosine similarity to find best match.
5. Return the corresponding answer.

## Conclusion
The chatbot successfully answers user questions using NLP techniques.
